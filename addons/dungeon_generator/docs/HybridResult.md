# HybridResult

**Inherits:** RefCounted

Object containing the data generated by a `HybridDungeonGenerator` run.

## Description

HybridResult stores the floor and wall tile positions, along with the physics-world room metadata and connectivity graph.

## Methods

| Returns | Method |
|---------|--------|
| Array | **get_rooms**() |
| Array | **get_links**() |

## Method Descriptions

### get_rooms() -> Array
Returns an array of Dictionaries, each representing a room from the physics phase.
Each dictionary contains:
- `id` (int): Unique identifier.
- `x`, `y` (float): Center position in **world units**.
- `w`, `h` (float): Dimensions in **world units**.
- `shape` (String): "rect" or "circle".
- `is_main` (bool): True if this room was used as a primary node in the graph.

### get_links() -> Array
Returns an array of Dictionaries representing the connectivity graph between rooms.
Each dictionary contains:
- `u`, `v` (int): Room IDs of the connected rooms.
- `is_mst` (bool): True if this link is part of the Minimum Spanning Tree (essential connection).

### get_floors() -> PackedVector2Array
Returns the positions of all walkable floor tiles in **grid coordinates**.

### get_walls() -> PackedVector2Array
Returns the positions of all wall tiles in **grid coordinates**.

### get_total_tiles() -> int
Returns the total number of non-empty tiles (floors + walls).

### get_grid_width() -> int
Returns the width of the grid used for generation.

### get_grid_height() -> int
Returns the height of the grid used for generation.

### get_tile_w() -> int
Returns the width scale factor. `x / tile_w` converts world X to grid X.

### get_tile_h() -> int
Returns the height scale factor. `y / tile_h` converts world Y to grid Y.
